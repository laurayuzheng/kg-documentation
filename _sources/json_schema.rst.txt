
JSON Schema 
============
The JSON schema for the preprocessing output is described as follows:

.. code-block:: JSON

   {
      "title": <str>,
      "authors": <list of str>,
      "emails": <list of str>,
      "institutions": <list of str>,
      "abstract": <str>,
      "references": <list of str>,
      "body_text": <str>                                 
   }
