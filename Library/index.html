Library Documentation¶

On this page, lower-level library functions are documented.

arXiv Scraping¶

This file bulk downloads from arXiv in two different formats: PDF and XML. A query is made to arXiv to retrieve metadata on related publications to keywords. This code can directly download related pdfs to a folder in the current directory or just retrieve the XML metadata.

Created by: Laura Zheng Date: Feb 2020

scraping.arXiv_scraper.build_query_list(keywords, conjunction=0, max_results=100)¶

Description: * uses build_query_url to create a list of query urls, with different combinations of keywords * this bulk download implementation is resistant to duplicates because of the way files are named

Parameters: * keywords --> list of keywords to include in search query * conjunction -->

  • default is 0, which is AND

  • 1 or anything nonzero would be OR

  • max_results --> default is 100. maximum number of publications retrieved during query

Returns: * urls --> returns a list of query URLs constructed from the keywords given

scraping.arXiv_scraper.build_query_url(keywords, conjunction=0, max_results=100)¶

Description: * makes a query to arXiv servers based on keywords and characteristics

Parameters: * keywords --> list of keywords to include in search query * conjunction -->

  • default is 0, which is AND

  • 1 or anything nonzero would be OR

  • max_results --> default is 100. maximum number of publications retrieved during query

Returns: * url --> the query URL that was built from the keywords

scraping.arXiv_scraper.download_pdfs(data_dir, urls)¶

Description: * downloads publications from arXiv based on earth science and machine learning keywords.

Parameters: * data_dir --> Path object pointing to data directory * urls --> list of arXiv query URLs

Returns: * nothing * side effect is downloaded publications to /papers/ folder under the current directory

scraping.arXiv_scraper.download_xml(data_dir, urls)¶

Description: * downloads XML file of metadata from arXiv based on earth science and machine learning keywords.

Parameters: * urls --> list of arXiv query URLs

Returns: * nothing * side effect is downloaded XML files to /arXiv-xml/ folder under the current directory
