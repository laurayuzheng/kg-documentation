Knowledge Graph Documentation¶

On this page, lower-level knowledge graph construction functions are documented.

SCiERC Evaluation Data Formatting¶

knowldgeB.scierc_format.format_scierc(dataDir)¶

Formats .txt files (corpus) in a directory to SCIERC data format, which is readable by SpERT for evaluation.

Arguments: dataDir - directory containing the text files

Returns: Nothing. Side effect is a data.json file that represents the data directory files as SCIERC dataset format.

Building the Knowledge Graph¶

knowldgeB.build_kg.build_kg(dataDir)¶

Parameters: - dataDir --> String indicating path of data, should be json file

Returns: - JSON object

knowldgeB.build_kg.get_span(start, end, tok_list)¶

Description: Gets the span in string form from output JSON data of SpERT.

Parameters: - start --> int, start index of span - end --> int, end index of span - tok_list --> token list

Returns: - String representation of span

knowldgeB.build_kg.valid_entity(s)¶

Description: Determines if a string entity is valid (e.g. starts with valid characters).

Parameters: - s --> string representing an entity

Returns: - Boolean --> valid or not

D3 Knowledge Graph Visualization¶

knowldgeB.kg_vis.copy_data(data_path, http_directory)¶

Copies the data from the processing directory to the HTTP visualization directory. Checks whether the data file exists and then checks if it is a valid JSON file.

Arguments:

data_path - the complete path to the data file, including the filename itself http_directory - the directory folder the data is copied to, not including the filename itself

Returns:

Nothing, side effect is file being copied.

knowldgeB.kg_vis.launch_kg_vis(directory)¶

Launches an HTTP server locally in order to visualize the knowledge graph data.

Arguments:

directory - Directory folder containing index.html

Returns:

Nothing. Should launch the visualization
